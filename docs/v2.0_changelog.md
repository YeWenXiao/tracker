# v2.0 开发日志 (v2.0-mipi 分支)

## 核心目标
去掉网络视频流 (RTSP)，使用 MIPI CSI 传视频以减少网络视频时延。

## 技术方案
- **视频源**: GStreamer + nvarguscamerasrc (Jetson Orin Nano)
- **硬件加速**: NVJPEG 解码、CUDA 预处理
- **延迟目标**: 从 RTSP 的 ~1000ms 首帧降到 ~50ms
- **兼容性**: 保留 --rtsp 作为 fallback
